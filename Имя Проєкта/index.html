About me

Hi! I'm SERGIO PETROVICH - HTML/CSS designer.

Welcome to my portfolio site. My name is Sergio. I live in Ukraine. I am engaged in HTML-layout of sites.

I have a fully cross-browser, adaptive and layout. To ensure that site layout requirements are met while working on a project, I use the Gulp project builder to automate the layout process, Less / Sass preprocessors for faster site styling and more flexible code, Pug templating engine, Pixel Perfect for layout that matches the provided layout " pixel by pixel”, BEM, Bootstrap, Smartgrid and media queries for responsive site layout. The dynamics of the site is organized by your own jQuery code or customized plugins.

At the end of the work, the layout is checked for display in different browsers on different devices, the code is checked by the validator for compliance with standards, and the identified errors are eliminated.
